# 個人成長目標管理系統 (Personal Growth CMS)

一個專為追蹤個人成長和量化目標進度而設計的內容管理系統。

## 功能特色

### 🎯 前台展示

- **進度視覺化**：清晰展示你與未來自己的差距
- **統計概覽**：總目標數、完成度、整體進度一目了然
- **進度圖表**：柱狀圖和餅圖展示目標狀態分布
- **個人對比**：目前的自己 vs 未來的自己對比分析
- **激勵訊息**：根據進度提供個人化的鼓勵訊息

### 💼 後台管理

- **目標管理**：創建、編輯、刪除個人目標
- **子目標系統**：將大目標分解為可管理的小步驟
- **進度追蹤**：勾選完成的子目標，自動計算整體進度
- **自我描述**：定義「目前的自己」和「未來的自己」
- **即時統計**：後台管理面板顯示目標統計

## 核心概念

### 📊 量化成長差距

- **0%**：目前的自己（起始狀態）
- **100%**：未來的自己（理想狀態）
- **進度計算**：當完成 n 個子目標中的 x 個時，與未來自己的差距為 `100 - (x/n * 100)%`

### 🎯 子目標系統

- 每個大目標可以分解為多個子目標
- 完成每個子目標都會縮小與未來自己的差距
- 例如：4 個子目標的目標，完成第 2 個時，進度為 50%，差距為 50%

## 技術架構

- **前端框架**：Next.js 15.3.4 + React 19
- **樣式系統**：Tailwind CSS 4
- **圖表庫**：Recharts（柱狀圖、餅圖）
- **圖標庫**：React Icons (Feather Icons)
- **表單處理**：React Hook Form
- **數據存儲**：localStorage（客戶端存儲）
- **UI 組件**：Headless UI

## 安裝與運行

### 前置需求

- Node.js 18+
- npm 或 yarn

### 安裝步驟

1. 克隆專案

```bash
git clone <repository-url>
cd next-cms
```

2. 安裝依賴

```bash
npm install
```

3. 啟動開發服務器

```bash
npm run dev
```

4. 開啟瀏覽器訪問 `http://localhost:3000`

### 建置部署

```bash
npm run build
npm start
```

## 使用指南

### 首次使用

1. 訪問首頁，點擊「載入範例資料」體驗系統功能
2. 或者點擊「建立目標」開始創建你的第一個目標

### 創建目標

1. 進入「後台管理」頁面
2. 點擊「新增目標」按鈕
3. 填寫目標資訊：
   - 目標標題和描述
   - 目前的自己（0%狀態描述）
   - 未來的自己（100%狀態描述）
   - 子目標列表（可動態添加/刪除）
4. 保存目標

### 追蹤進度

1. 在後台管理中勾選已完成的子目標
2. 系統自動計算整體進度
3. 前台即時更新進度展示

## 功能截圖

### 前台展示
